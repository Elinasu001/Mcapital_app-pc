크기 조정을 위한 핼퍼 클래스
  • width
    • px
      • 클래스 이름: .ei-w숫자
      • 5~1000까지 5px 간격
    • %
      • 클래스 이름: .ei-w-p숫자
      • 1~100까지 1% 간격
      • width 33.3% : .ei-w-p333, width 66.6% : .ei-w-p666
  • height
    • 클래스 이름: .ei-h숫자
    • 5~1000까지 5px 간격
  • padding
    • 클래스 종류
      • .ei-p숫자 : padding 전체
      • .ei-py숫자 : padding-top, padding-bottom
      • .ei-px숫자 : padding-left, padding-right
      • .ei-pt숫자 : padding-top
      • .ei-pb숫자 : padding-bottom
      • .ei-pl숫자 : padding-left
      • .ei-pr숫자 : padding-right
    • 5~100까지 5px 간격
  • margin
    • 클래스 종류
      • .ei-m숫자 : margin 전체
      • .ei-my숫자 : margin-top, margin-bottom
      • .ei-mx숫자 : margin-left, margin-right
      • .ei-mt숫자 : margin-top
      • .ei-mb숫자 : margin-bottom
      • .ei-ml숫자 : margin-left
      • .ei-mr숫자 : margin-right
    • 5~100까지 5px 단위
★ 기본 컴포넌트에 적용 시 모양이 변형되므로 콘텐츠의 영역 조정 시에만 사용 권장
★ input text, select에는 width 변경하는 클래스만 사용 가능
